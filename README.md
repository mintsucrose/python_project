## Описание программы

**Notes App** — это оконное приложение на Python/Tkinter, которое позволяет:

- создавать заметки с заголовком и текстом;
- сохранять их в файл `notes.json`;
- просматривать список всех заметок и содержимое каждой;
- редактировать и удалять заметки.

Программа решает проблему быстрой организации записей: все заметки хранятся в одном месте, доступны сразу после запуска, не требуют интернета или регистрации.
Интерфейс интуитивно понятен и не перегружен лишними элементами.

---

## Требования для запуска

- Python **3.10** или выше
- Стандартная библиотека **tkinter** (входит в установку Python)
- Сторонние библиотеки **не требуются**

---

## Как запустить

```bash
git clone https://github.com/mintsucrose/python_project.git
cd python_project
python main.py
```
После запуска откроется окно «Заметки».

## Пример работы

### Добавление заметки

1. Ввести заголовок (например, «Покупки»)
2. Ввести текст (например, «Купить хлеб, молоко, яйца»)
3. Нажать **💾 Сохранить**
