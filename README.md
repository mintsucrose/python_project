## Описание программы

**Notes App** — это оконное приложение на Python/Tkinter, которое позволяет:

- создавать заметки с заголовком и текстом;
- сохранять их в файл `notes.json`;
- просматривать список всех заметок и содержимое каждой;
- редактировать и удалять заметки.

Программа решает проблему быстрой организации записей: все заметки хранятся в одном месте, доступны сразу после запуска, не требуют интернета или регистрации.
Интерфейс интуитивно понятен и не перегружен лишними элементами.

## Требования для запуска

- Python **3.10** или выше
- Стандартная библиотека **tkinter** (входит в установку Python)
- Сторонние библиотеки **не требуются**

## Как запустить

```bash
git clone https://github.com/mintsucrose/python_project.git
cd python_project

```
После запуска откроется окно «Заметки».

## Пример работы

### Добавление заметки

1. Ввести заголовок (например, «Покупки»)
2. Ввести текст (например, «Купить хлеб, молоко, яйца»)
3. Нажать **💾 Сохранить**

Результат:

```text
✅ Заметка «Покупки» добавлена
```
### Файл notes.json:
[
    {
        "title": "Покупки",
        "content": "Купить хлеб, молоко, яйца",
        "date": "05.05.2026 15:30"
    }
]
### Просмотр содержимого

Кликнуть по заметке в списке — справа отобразится полный текст.

### Редактирование

1. Выбрать заметку в списке
2. Нажать **✏️ Редактировать**
3. Изменить заголовок или текст
4. Нажать **🔄 Обновить**

### Удаление

1. Выбрать заметку в списке
2. Нажать **🗑 Удалить**
3. Подтвердить действие

Результат:

```text
🗑 Заметка «Покупки» удалена
```
## Тестирование

| Тип теста | Пример | Результат |
|-----------|--------|-----------|
| Позитивный | Ввод заголовка «Встреча», текста «Завтра в 18:00» | ✅ Заметка добавлена |
| Негативный | Пустой заголовок при сохранении | ❌ Предупреждение: «Введите заголовок заметки!» |
| Негативный | Пустой текст при сохранении | ❌ Предупреждение: «Введите текст заметки!» |
| Граничный | Удаление заметки и отмена в диалоге | ✅ Заметка не удалена |
