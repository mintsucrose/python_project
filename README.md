# Weather Diary — Дневник погоды

**Автор:** Глухих Александра
**Дата сдачи:** 05.05.2026

---

## Описание программы

GUI-приложение для ведения дневника погоды. Позволяет добавлять записи с датой, температурой, описанием и осадками. Данные сохраняются в JSON. Реализована фильтрация по дате и температуре.

---

## Требования для запуска

- Python 3.10+
- Tkinter (встроен)

---

## Как запустить

```bash
git clone https://github.com/mintsucrose/python_project.git
cd python_project
python main.py
```
---

## Пример работы

### Добавление записи

Ввод: Дата 2026-05-15, Температура 22.5, Описание Солнечно, Осадки Нет

Результат:

```text
Результат: ✅ Запись добавлена
```
### Фильтрация:
```
По дате: 2026-05-15 → показаны записи за этот день

По температуре: 10 → показаны записи ≥ 10°C
```
## Тестирование

| Тип теста | Пример | Результат |
|-----------|--------|-----------|
| Позитивный | Дата: 2026-05-15, t°C: 22.5, Описание: Солнечно | ✅ Добавлено |
| Негативный | Дата: abc | ❌ Ошибка формата даты |
| Негативный | t°C: abc | ❌ Температура должна быть числом |
| Негативный | Пустое описание | ❌ Описание не может быть пустым |
| Граничный | Фильтр по несуществующей дате | ❌ Записи не найдены |
